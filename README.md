# weatherApp
App with NW.js and NWBuilder to create OSX, Linux and Windows desktop app from JS


## Getting Started

### 1) Clone the GIT project
 ```
 https://github.com/michaelmoney/weatherApp.git
 ```
 
```
 git@github.com:michaelmoney/weatherApp.git
 ```

### 2) In terminal run:
```
npm install
```

### 3a) Dev mode
```
grunt serve
```

### 3b) Build package
```
grunt build
```
